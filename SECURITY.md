# 🔐 Security Policy

## 📌 Supported Versions

Currently, we actively maintain and provide security updates for the latest version of TaskMaster.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅        |
| Older Versions | ❌ |

---

## 🚨 Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

📧 Email: arenaxenon@gmail.com  

### What to include:
- Description of the vulnerability
- Steps to reproduce the issue
- Possible impact
- Screenshots or proof (if applicable)

---

## ⏱ Response Time

- Initial response within **24–48 hours**
- Fix or update will be provided as soon as possible based on severity

---

## 🤝 Responsible Disclosure

Please do not publicly disclose the issue until it has been reviewed and resolved.  
We appreciate your cooperation in keeping the project and its users safe.

---

Your efforts help improve the security of TaskMaster
