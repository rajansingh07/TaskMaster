# TaskMaster

A simple and responsive **To-Do List app** built with **React** and **React Bootstrap**. TaskMaster allows users to add, filter, and mark tasks as completed with a clean and modern interface.

## Features

- Add new tasks with title, description, and priority.
- Mark tasks as completed.
- Filter tasks by status (All, Completed, Pending).
- Responsive layout for desktop and mobile.
- Priority badges with color coding (High, Medium, Low).
- Smooth hover effects and interactive UI.

## Tech Stack

- **Frontend:** React, React Bootstrap, CSS
- **Routing:** React Router
- **Styling:** Custom CSS + Bootstrap

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/taskmaster.git
   cd taskmaster
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the app locally:

   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
