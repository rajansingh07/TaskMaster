# Contributing to TaskMaster

Thanks for your interest in contributing to TaskMaster 🚀  
We welcome contributions from everyone.

## 📌 How to Contribute

1. Fork the repository
2. Clone your fork - git clone https://github.com/rajansingh07/TaskMaster.git

Create a new branch

git checkout -b feature/your-feature-name
Make your changes

Commit your changes

git commit -m "Add: your feature description"

Push to your branch

git push origin feature/your-feature-name
Create a Pull Request
🛠 Contribution Guidelines
Write clean and readable code
Follow proper naming conventions
Keep commits meaningful and short
Test your changes before submitting
Avoid unnecessary changes
Reporting Issues

If you find a bug:

Clearly describe the issue
Add steps to reproduce
Include screenshots (if possible)
💡 Feature Requests

We welcome ideas!
Open an issue and describe your feature clearly.

📄 Code of Conduct

Be respectful and supportive.
We aim to maintain a friendly environment for everyone.

Thank You

Your contributions help improve TaskMaster
